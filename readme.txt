Gutenberg Photo Feed Gallery - the easiest way to add instagram photo feed without api or access token.

== Description ==

A most flexible guteberg block to add instagram photo gallery. Display instagram feed from your instagram account without any hassle like api key or access token. It is quite easy and simple just put your instagram profile username on setting and you are ready to go. 


= How to Install Photo Feed Gallery? =

1. Go to your **Dashboard**
2. Opt-in to **Plugins > Add New** 
3. Search for **Photo Feed Gallery**
4. Click install and Activate


== Frequently Asked Questions ==

= Is this plugin do any api request to instagram? =
Yes, it is using [jQuery Instagram Feed](https://github.com/jsanahuja/jquery.instagramFeed) to make a simple call to get your instagram data. But this don't need any api key or access token for this. You just need to give the username and that's it. 

= Does this need access token or api key? =
Absolutely not, until instagram restrict data to call this way.
